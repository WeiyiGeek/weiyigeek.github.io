Oracleе‘Ҫд»ӨеҸӮж•°дёҖи§ҲпјҲеҫ…е®ҢжҲҗпјү

2019-06-18
  • Database

[TOC]

oracleж•°жҚ®еә“зҡ„жқғйҷҗзі»з»ҹеҲҶдёәзі»з»ҹжқғйҷҗдёҺеҜ№иұЎжқғйҷҗгҖӮзі»з»ҹжқғйҷҗ( database system privilege )еҸҜд»Ҙи®©з”ЁжҲ·жү§иЎҢзү№е®ҡзҡ„е‘Ҫд»ӨйӣҶгҖӮдҫӢеҰӮпјҢcreate tableжқғйҷҗе…Ғи®ёз”ЁжҲ·еҲӣе»әиЎЁпјҢgrant any privilege жқғйҷҗе…Ғи®ёз”ЁжҲ·жҺҲдәҲд»»дҪ•зі»з»ҹжқғйҷҗгҖӮеҜ№иұЎжқғйҷҗ( database object privilege )еҸҜд»Ҙи®©з”ЁжҲ·иғҪеӨҹеҜ№еҗ„дёӘеҜ№иұЎиҝӣиЎҢжҹҗдәӣж“ҚдҪңгҖӮдҫӢеҰӮdeleteжқғйҷҗе…Ғи®ёз”ЁжҲ·еҲ йҷӨиЎЁжҲ–и§Ҷеӣҫзҡ„иЎҢпјҢselectжқғйҷҗе…Ғи®ёз”ЁжҲ·йҖҡиҝҮselectд»ҺиЎЁгҖҒи§ҶеӣҫгҖҒеәҸеҲ—(sequences)жҲ–еҝ«з…§(snapshots)дёӯжҹҘиҜўдҝЎжҒҜгҖӮ

жҜҸдёӘoracleз”ЁжҲ·йғҪжңүдёҖдёӘеҗҚеӯ—е’ҢеҸЈд»Ө,е№¶жӢҘжңүдёҖдәӣз”ұе…¶еҲӣе»әзҡ„иЎЁгҖҒи§Ҷеӣҫе’Ңе…¶д»–иө„жәҗгҖӮoracleи§’иүІ(role)е°ұжҳҜдёҖз»„жқғйҷҗ(privilege)(жҲ–иҖ…жҳҜжҜҸдёӘз”ЁжҲ·ж №жҚ®е…¶зҠ¶жҖҒе’ҢжқЎд»¶жүҖйңҖзҡ„и®ҝй—®зұ»еһӢ)гҖӮз”ЁжҲ·еҸҜд»Ҙз»ҷи§’иүІжҺҲдәҲжҲ–иөӢдәҲжҢҮе®ҡзҡ„жқғйҷҗпјҢз„¶еҗҺе°Ҷи§’иүІиөӢз»ҷзӣёеә”зҡ„з”ЁжҲ·гҖӮдёҖдёӘз”ЁжҲ·д№ҹеҸҜд»ҘзӣҙжҺҘз»ҷе…¶д»–з”ЁжҲ·жҺҲжқғгҖӮ

дёҖгҖҒеҲӣе»әз”ЁжҲ·

oracleеҶ…йғЁжңүдёӨдёӘе»әеҘҪзҡ„з”ЁжҲ·пјҡsystemе’ҢsysгҖӮз”ЁжҲ·еҸҜзӣҙжҺҘзҷ»еҪ•еҲ°systemз”ЁжҲ·д»ҘеҲӣе»әе…¶д»–з”ЁжҲ·пјҢеӣ дёәsystemе…·жңүеҲӣе»әеҲ« зҡ„з”ЁжҲ·зҡ„ жқғйҷҗгҖӮ еңЁе®үиЈ…oracleж—¶пјҢз”ЁжҲ·жҲ–зі»з»ҹз®ЎзҗҶе‘ҳйҰ–е…ҲеҸҜд»ҘдёәиҮӘе·ұе»әз«ӢдёҖдёӘз”ЁжҲ·гҖӮ

иҜӯжі•[еҲӣе»әз”ЁжҲ·]пјҡ create user з”ЁжҲ·еҗҚ identified by еҸЈд»Ө[еҚіеҜҶз Ғ]пјӣ

дҫӢеӯҗпјҡ create user test identified by test;

иҜӯжі•[жӣҙж”№з”ЁжҲ·]: alter user з”ЁжҲ·еҗҚ identified by еҸЈд»Ө[ж”№еҸҳзҡ„еҸЈд»Ө];

дҫӢеӯҗпјҡ alter user test identified by 123456;

дәҢгҖҒеҲ йҷӨз”ЁжҲ·

иҜӯжі•пјҡdrop user з”ЁжҲ·еҗҚ;

дҫӢеӯҗпјҡdrop user test;

иӢҘз”ЁжҲ·жӢҘжңүеҜ№иұЎпјҢеҲҷдёҚиғҪзӣҙжҺҘеҲ йҷӨпјҢеҗҰеҲҷе°Ҷиҝ”еӣһдёҖдёӘй”ҷиҜҜеҖјгҖӮжҢҮе®ҡе…ій”®еӯ—cascade,еҸҜеҲ йҷӨз”ЁжҲ·жүҖжңүзҡ„еҜ№иұЎпјҢз„¶еҗҺеҶҚеҲ йҷӨз”ЁжҲ·гҖӮ

иҜӯжі•пјҡ drop user з”ЁжҲ·еҗҚ cascade;

дҫӢеӯҗпјҡ drop user test cascade;

дёүгҖҒжҺҲжқғи§’иүІ

oracleдёәе…је®№д»ҘеүҚзүҲжң¬пјҢжҸҗдҫӣдёүз§Қж ҮеҮҶи§’иүІпјҲroleпјү:connect/resourceе’Ңdba.

пјҲ1пјүи®Іи§Јдёүз§Қж ҮеҮҶи§’иүІпјҡ

1гҖӢ. connect role(иҝһжҺҘи§’иүІ)

вҖ“дёҙж—¶з”ЁжҲ·пјҢзү№жҢҮдёҚйңҖиҰҒе»әиЎЁзҡ„з”ЁжҲ·пјҢйҖҡеёёеҸӘиөӢдәҲд»–д»¬connect role.

вҖ“connectжҳҜдҪҝз”Ёoracleз®ҖеҚ•жқғйҷҗпјҢиҝҷз§ҚжқғйҷҗеҸӘеҜ№е…¶д»–з”ЁжҲ·зҡ„иЎЁжңүи®ҝй—®жқғйҷҗпјҢеҢ…жӢ¬select/insert/updateе’ҢdeleteзӯүгҖӮ

вҖ“жӢҘжңүconnect role зҡ„з”ЁжҲ·иҝҳиғҪеӨҹеҲӣе»әиЎЁгҖҒи§ҶеӣҫгҖҒеәҸеҲ—пјҲsequenceпјүгҖҒз°ҮпјҲclusterпјүгҖҒеҗҢд№үиҜҚ(synonym)гҖҒеӣһиҜқпјҲsessionпјүе’Ңе…¶д»– ж•°жҚ®зҡ„й“ҫпјҲlinkпјү

2гҖӢ. resource role(иө„жәҗи§’иүІ)

вҖ“жӣҙеҸҜйқ е’ҢжӯЈејҸзҡ„ж•°жҚ®еә“з”ЁжҲ·еҸҜд»ҘжҺҲдәҲresource roleгҖӮ

вҖ“resourceжҸҗдҫӣз»ҷз”ЁжҲ·еҸҰеӨ–зҡ„жқғйҷҗд»ҘеҲӣе»әд»–д»¬иҮӘе·ұзҡ„иЎЁгҖҒеәҸеҲ—гҖҒиҝҮзЁӢ(procedure)гҖҒи§ҰеҸ‘еҷЁ(trigger)гҖҒзҙўеј•(index)е’Ңз°Ү(cluster)гҖӮ

3гҖӢ. dba role(ж•°жҚ®еә“з®ЎзҗҶе‘ҳи§’иүІ)

вҖ“dba roleжӢҘжңүжүҖжңүзҡ„зі»з»ҹжқғйҷҗ

вҖ“еҢ…жӢ¬ж— йҷҗеҲ¶зҡ„з©әй—ҙйҷҗйўқе’Ңз»ҷе…¶д»–з”ЁжҲ·жҺҲдәҲеҗ„з§Қжқғйҷҗзҡ„иғҪеҠӣгҖӮsystemз”ұdbaз”ЁжҲ·жӢҘжңү

пјҲ2пјүжҺҲжқғе‘Ҫд»Ө

иҜӯжі•пјҡ grant connect, resource to з”ЁжҲ·еҗҚ;

дҫӢеӯҗпјҡ grant connect, resource to test;

пјҲ3пјүж’Өй”Җжқғйҷҗ

иҜӯжі•пјҡ revoke connect, resource from з”ЁжҲ·еҗҚ;

еҲ—еӯҗпјҡ revoke connect, resource from test;

еӣӣгҖҒеҲӣе»ә/жҺҲжқғ/еҲ йҷӨи§’иүІ

йҷӨдәҶеүҚйқўи®ІеҲ°зҡ„дёүз§Қзі»з»ҹи§’иүІвҖ”-connectгҖҒresourceе’ҢdbaпјҢз”ЁжҲ·иҝҳеҸҜд»ҘеңЁoracleеҲӣе»әиҮӘе·ұзҡ„roleгҖӮз”ЁжҲ·еҲӣе»әзҡ„roleеҸҜд»Ҙз”ұиЎЁжҲ–зі»з»ҹжқғйҷҗжҲ–дёӨиҖ…зҡ„з»„еҗҲжһ„жҲҗгҖӮдёәдәҶеҲӣе»әroleпјҢз”ЁжҲ·еҝ…йЎ»е…·жңүcreate roleзі»з»ҹжқғйҷҗгҖӮ

1гҖӢеҲӣе»әи§’иүІ

иҜӯжі•пјҡ create role и§’иүІеҗҚ;

дҫӢеӯҗпјҡ create role testRole;

2гҖӢжҺҲжқғи§’иүІ

иҜӯжі•пјҡ grant select on class to и§’иүІеҗҚ;

еҲ—еӯҗпјҡ grant select on class to testRole;

жіЁпјҡзҺ°еңЁпјҢжӢҘжңүtestRoleи§’иүІзҡ„жүҖжңүз”ЁжҲ·йғҪе…·жңүеҜ№classиЎЁзҡ„selectжҹҘиҜўжқғйҷҗ

3гҖӢеҲ йҷӨи§’иүІ

иҜӯжі•пјҡ drop role и§’иүІеҗҚ;

дҫӢеӯҗпјҡ drop role testRole;

жіЁпјҡдёҺtestRoleи§’иүІзӣёе…ізҡ„жқғйҷҗе°Ҷд»Һж•°жҚ®еә“е…ЁйғЁеҲ йҷӨ

зӣ‘еҗ¬зҡ„еёёз”Ёе‘Ҫд»Ө

lsnrctl stop [listener_name]

lsnrctl status [listener_name]
