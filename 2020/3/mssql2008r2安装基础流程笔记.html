SQL Server 2008 R2的核心服务器版本说明:

SQL Server 2008 R2的硬件和软件要求:

  • 系统要求:建议在Windows Server 2008或Windows 7及以上在Server 2012 R2版本以下(注意Server 2019不支持SQL Server 2008 R2)
  • 磁盘分区:建议采用NTFS文件格式
  • 开发工具包:.NET Framework 3.5
  • 系统位数: 建议X64
  • 内存: 最小:1 GB / 推荐 4 GB或更多 (此处以标准版64位为例最大支持64 GB)
