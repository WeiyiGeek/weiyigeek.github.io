处理器|CPU

CPU各种重要参数:

  • 核心频率(CPU时钟频率):反应工作速度一个最直观的参数,频率越高单位时间内CPU执行的指令数就越多，且分为基础频率（CPU正常工作的最低频率）和加速频率（超频过后的最高频率）;
  • 制作工艺:CPU中一个晶体管门电路的大小,从intel第一款CPU的10微米的Intel 4004开始,到AMD推出的7nm Zen2系列；电脑CPU的制作工艺已经缩小了1400多倍;一般来所工业越小功耗和发热就会更小但同积体放下的晶体管数就越多性能就越强;
  • 核心/线程数:有几核心决定了同一时间内可以处理多个任务;
  • CPU缓存:现在基本是三级缓存由于CPU速度远高于内存运行速度保证内存不拖CPU内存的后退才加入缓存,CPU一级与二级缓存差别一般不大主要对比不同处理器的三级缓存即可，对于游戏缓存大小区别不是很大但是进行视频渲染的时候就能体现出差距了,当然缓存是越大越好;

CPU制作过程:

  • 1.制作原材料就是沙子中的(SiO2 + C )进行高温还原硅再经过氯化与还原得到村都极高的硅单质;
  • 2.再使用单晶炉拉制出单晶硅,然后经过切割和打磨就变成了晶圆，随后进行掺杂以增加半导体的导电性;
  • 3.把感光遮罩覆盖到晶圆上用光刻机进行对晶圆烙印，被紫外线光照的地方感光遮罩会变成水溶性物质；
  • 4.清洗后进行刻蚀然后就可以再晶圆上刻出几十亿个晶体管,再附上阻挡层并进行多次的清洗;
  • 5.最后再表面覆盖铜打磨并进行连线 / 切割 /封装;

CPU原理了解

  • 二进制进行逻辑运算，1和0也就是晶体管的开和关(与或门)；将数亿个晶体管组合在一起就是CPU当中最重要的一个部件Arithmetic and Logic Unit(算术逻辑单元);
  • Control Unit(控制单元)进行指令译码，其次还有存储指令与数据高速缓存，还有与外部连线的各种总线;

问:CPU中的Hz数代表什么?为啥要超频使用?

答:代表了单位时间内CPU执行指令数，Hz越大执行的指令数就越大，然后您懂得;

问:核心数和线程数是不是越多越好?

一般的游戏对超多核心和线程CPU优化不是很好，比如线程撕裂者;
一般来说4到8核心以及4到16线程是比较主流的游戏平台的选择;

问:CPU盒装与散装的区别?

盒装:怕上当官网购买中文原盒CPU,该CPU自带原装散热风扇;包装/风扇和CPU上的SN码一致,提供质保的;
散装:通过海淘非正规渠道流入大陆的原装正品CPU，比中文原盒便宜三码一致，但是不提供保修;
散片:各种拆机件或者非正规渠道的货所以它更加的便宜，没有包装没有散热（水有点深DIY玩家玩家）;

主板|Mainboard

Mainboard重要参数:

  • 1.芯片组:H310 / B360 / Z370 / Z390(CPU插槽/主板接口类型为LGA1151), B350 / B450 / X370 / X470 (CPU插槽/AM4接口) , X399 (CPU插槽/TR4接口)

  • 2.版型选择:EATX / ATX / M - ATX / ITX[NAS常用] (主板从大到小)，其他版型的价格基本与其大小成为正比，越大的板型相当于拥有了更多的扩展接口,芯片组规格也就越高;

  • 3.主板频率:即主板最大支持的频率
  • 4.主板内存:DDR3 / DDR4 (带L表示笔记本使用) 以及最大内存数

  • 5.PCI-e插槽:一般分为四种 x1(声卡) / x4（网卡 / NVMe协议M.2固态） / x8 / x16(显卡)， 每一条PCIE通道可以提供算共享四百兆的速率;

  • 6.USB协议

主板零件了解:(以超微Z390举例)

  • 供电: 8pin供电(高端主板采用8+8pin两个接口,自然支持更多的供电相数)，现阶段主板基本采用数字供电(该部分为贴片电容/固态电容/MOSFET金氧半场效应晶体管/议题电感/PWN主控组成)，理论上一相供电按25w计算四相就是100w，与其对应的CPU参数为TDP（注意主板供电一定要比TDP高否则CPU不能正常工作）;
  • CPU插槽:两种接口LGA封装(平面网格阵列封装) / FGA封装(插针网格阵列封装) ，其周围会留4PIN的风扇插槽而高端主板则有两道三个插槽，用来安装更高端的多风扇散热器;
  • 内存插槽:中高端主板一般会配有4个内存插槽 / 地段主板则只有2个内存插槽,而特色的X299 与 x399则支持8根内存;
  • PCI-e插槽:芯片组规格越高，支持的PCI-e总线数就越多，插槽数也就越多;
  • CR2032电池纽扣:记录主板在断电状态下BIOS当中存储的设置信息，当主板出现兼容性问题时候，进行放电则使BIOS回归初始设置;
  • 南桥 / 北桥 : 前者控制外部输入输出设备，后者控制内部设备CPU/内存
  • USB3.x及SATA接口 / 网口规格 (常规千兆) / 音频IO自带同轴光纤输出
  • COM1 : 串口通常连接工业设备
  • tpm : 接口用来进加密
  • JFP1 :接口用于连接前置面板开关指示灯，一般测试主板是否可以正常工作在装入机箱前需要进行点亮查看，在经典的9针跳线左上角的两条用螺丝刀轻轻触碰一下;

显卡|GraphicsCard

显卡重要参数:

  • 1.CUDA核心数:Compute Unified Device Architecutre统一计算框架的缩写,可以理解成为比CPU核心小得多的核心堆到一起组成GPU(擅长大量数据的并行运算)，也就是说核心数越多同时进行并行计算数量就越高，显卡的性能就越强;
  • 2.核心频率:频率越高单个CUDA核心运算速度就越快
  • 3.显存大小:即显存容量一般用来存储即将或已经被GPU计算的数据,如果显存不够溢出的而数据就会转到内存当中，但是内存通道的速度远远慢于GPU到显存的速度（这就是掉帧），所以拥有足够的显存容量是非常重要的;
  • 4.显存位宽:拥有越大的显存位宽同时传输的数据量就会变大,存储效率就会提高;
  • 5.显存频率:与内存频率一样显存频率反应的是显存颗粒内硅晶震动频率,频率越高显存工作效率就越高;
  • 6.流处理器数:流处理器单元是统一架构GPU内通用标量着色器的称谓。 流处理单元直接影响处理能力，因为流处理单元是显卡的核心。
  • 7.显存工艺:从GDDR到现在的GDDR6和HBM2,显存规格越高支持的大小 位宽和频率就会相应的提高;

问:英伟达(NVIDIA)与AMD显卡的抉择?
答:同等价位的AMD显卡参数性能比英伟达显卡略好一些，对于游戏方面的性能都是差不多的，但是AMD的设计功耗要高于同级别英伟达显卡所以对供电需求高一些；

问:什么是公版/非公版显卡?

公版:NVIDIA 或者 AMD 直接卖给消费者的显卡;特点:有料工作稳定，但是超频空间不大，扇热帅但是效果较弱，一般买它的人都是改分体水冷或者组slicf消费者；
非公版:则是国内各大厂商拿着英伟达或者AMD的核心，自己做的显卡:可玩性比较强会自己设置PCB和散热器，所以供电和扇热都会比公版强出不少,出厂核心与显存频率都比公版高一些;

问:如何挑选显卡的品牌?

使用 NVIDIA 芯片厂商:ASUS / MSI / EVGA / GIGABYTE / ZOTAC(索泰) / inno3D / 影驰 / Colorful 等都是英伟达的AIC,拥有自己的研发线能够独立设计PCB,所以他们高端系列显卡性能较强，不建议铭瑄 / 翔升 / 盈通之类的品牌;
使用 AMD 芯片厂商:讯景(XFX)/ 蓝宝石(SAPPHIRE) / 迪兰)(ATALAND)

描述:不管是PC还是笔记本再安装散热器时候都必须再CPU(GPU)和散热器之间涂上一层散热硅胶，才能获得更好的散热效果;

问:为什么要有散热硅胶?

答:再风扇与CPU之间，由于金属的导热性是非常好的，比如铜的导热系数是377W/m.k,但是不管现在生产工业多么高超都不能保障金属表面绝对的光滑并且安装时候也不能包装两者之间完全紧密接触，那么中间会有少量的空气（导热系数:0.025W/m.k）比较低会影响散热器的导热效率;
硅脂:就是综合导热系数/性价比以及适用性比较好的选择,纯硅脂的产品绝大多数是绝缘的不用担心弄到主板电路上;

问:如何涂抹硅脂?

答:均匀涂抹一层完整覆盖尽量薄一些,本身就是为了填充空袭太厚了反而影响导热;

推荐选择

  • 酷冷至尊（凉膏） 1 W/m.k
  • 超频三 : A1 7.5 W/m.k
  • ARCTIC : MX4 8.5 W/m.k
  • 液体金属 : 128W/m.k 以上 (高端玩家)价格高一般给服务器CPU涂抹，而且操作中非常需要小心，万一滴落再主板裸露电路上会导致短路(急救办法:酒精液荼洗);

补充事项:

  • 金银的导热系数高可以达到300~400W/m.k;
  • 需要注意如果购买的硅脂是含有金属的那就会导电，不要弄到了CPU主板上;
  • 散热硅胶片不能用于CPU上一般用于笔记本显存/主板芯片上等发热高散热要求不高的地方，作为辅助散热填充物;
